# SkillSprout MoodScape 🌈

An Emotion-Based Experience Generator. Pick your mood and get a matching message, music suggestion, and mini challenge.

## How to Use

1. Open the site
2. Select your mood
3. Click **Generate My Experience**

- HTML

- JavaScript
